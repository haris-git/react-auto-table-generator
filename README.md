This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app). It can receives some data as an array of javascript objects and view them inside a table. The columns of the table are calculated during runtime. The table is constructed with the "Material UI".

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000/?key=products](http://localhost:3000/?key=products) to view it in the browser.
